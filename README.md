
# LinkExt

### A python Script for automatically collect links from a web page. <br>

<br>

## Usage
```yaml
apt-get update
apt-get upgrade
pkg install git
pkg install python3
pip install -r requirements.txt
git clone https://github.com/thenurhabib/linkext
cd linkext
python3 run.py`
```
